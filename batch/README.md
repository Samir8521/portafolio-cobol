Procesamiento de Movimientos Bancarios (Batch COBOL)
Este módulo batch procesa un archivo de movimientos bancarios en formato texto, validando cada registro y generando un reporte final con los datos correctos.
Descripción del programa
El programa MOVIMIENTOS-BANCARIOS realiza:
- Lectura secuencial de un archivo de movimientos
- Validación de campos (cuenta, fecha, tipo de movimiento, monto)
- Control de registros válidos e inválidos
- Generación de un archivo de salida formateado
- Conteo total de registros procesados
